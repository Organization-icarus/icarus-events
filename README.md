# Icarus Events (Event Lottery System)

A simple Android + Firestore prototype for an event lottery/waitlist system. Users can join event waitlists via QR code, organizers can manage events and entrants, and administrators can moderate content.

## Tech Stack
- Android (Java)
- Firebase / Firestore
- GitHub for collaboration (branches + PRs)

## Repository Structure
- `Icarus/` — Android Studio project (source code)
- `doc/` — Project documentation (meeting minutes, mockups/storyboards, backlog, UML, etc.)
